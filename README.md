## 📌 Project Overview

Managing property operations requires continuous monitoring of revenue, occupancy, bookings, and overall portfolio health.

This project provides an interactive analytics platform that enables users to:

* Upload CSV and Excel datasets
* Assess data quality
* Monitor key performance indicators (KPIs)
* Visualize revenue and occupancy trends
* Rank properties by performance
* Calculate property health scores
* Generate executive summaries for decision-makers

The platform is designed as a practical business intelligence solution for property managers, operations teams, and analysts.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ZiaMehmood1996/ai-property-operations-platform.git
```

Move into the project folder:

```bash
cd ai-property-operations-platform
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

### Windows

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
